模型配置文件 models.json

选择一个 API Key，即可生成已配置好的 models.json，下载后替换 WorBuddy 配置目录中的原文件即可使用。

第一步：备份原配置文件

配置文件位于 ~/.workbuddy/models.json 备份将另存为 models-年-月-日-时-分.json。

第二步：下载新配置文件


          
配置文件中会直接包含所选 Key 的完整密钥，请妥善保管。

          
第三步：重启 WorBuddy

          
用新下载的 models.json 替换配置目录中的原文件后，退出 WorBuddy 并重新打开，新配置即生效。
